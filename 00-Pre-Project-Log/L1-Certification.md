# L1 Certification Build

Find the full project here (repo link)

## Overview
Stratus is an high power cable scartch build, desinged to be cheap (not including the components I already had such as flight computers and GPS)
in order to acheive my L1 Certification.

## Flight Config
Stratus L1 Version
- Motor: CTI pro38-2G H143-10
- Airframe: 1050mm x 75mm Cardboard
- Nose Cone: ABS 3:1 Ogive
- Fins: Trapizodial 3.6mm Ply
- Recovery: Motor eject dual deploy, 24" TFR main, 18" TFR drouge
- Avionics: Eggfinder Mini GPS, Blue Raven altimeter, nose cone AV bay
- Stability: 2.05 cal / 15%
- Expected Apogee : 2150ft
- Max Velocity: 187 m/s (mach 0.554)
- Max Acceleration 14 G
